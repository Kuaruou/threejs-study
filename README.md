*簡略說明

2. 裝置滾動滑鼠可以縮放圖片，且圖片有旋轉效果。

*技術
<p>
  <code>Three.js</code>
  <code>OrbitControls</code>
  <code>GLTFLoader</code>
</p>

*頁面部屬提示: 圖片路徑要為 ./ 開頭而不能為 / 開頭。
